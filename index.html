Temporal Analysis of Mobile Application Usage Habits and Patterns

0: 0 am

DESCRIPTION

This project delves into smartphone usage habits, exploring applications and duration of use. It presents four key visualizations for valuable insights. The first employs a dynamic bubble chart with an interactive clock, showcasing temporal app usage evolution. The second features a heatmap and connected barchart, allowing users to track app usage trends over time. The third provides a comparative analysis between weekdays and weekends, revealing distinct preferences and the fourth offers a comprehensive 7-month overview, highlighting the most used applications during this periode
